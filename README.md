# Simple Chatbot
Personal Project – 2024

## Description
A rule-based chatbot built in Java that simulates basic conversation. 
The chatbot responds to user input using switch/case logic and returns 
randomized responses to keep interactions varied.

## What I Built
- Implemented conversation flow using switch/case logic in Java
- Used randomized response arrays to simulate more natural replies
- Handled edge cases including an exit condition to end the conversation cleanly
- Built independently as a self-directed learning project to practice core Java concepts

## Technologies Used
- Java
- Eclipse IDE

## How to Run
1. Clone the repository
2. Open the project in Eclipse or any Java IDE
3. Run SimpleChatbot.java
4. Type messages in the console to interact with the chatbot
5. Type `exit` to end the conversation
